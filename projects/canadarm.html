LEGO Canadarm

A LEGO Canadarm prototype with wheel-motor joint control, colour-sensor height adjustment, and force-sensitive gripper activation — three sensor modalities in one arm.

Arduino C++ Control Systems Sensor Fusion LEGO Robotics Force Sensing
Purpose

Why I built this.

Inspired by the iconic Canadarm on the International Space Station, this project built a LEGO-based robotic arm that uses three different sensing modalities to control its three main functions — a more sophisticated control scheme than most student robot projects attempt.

Process

How I built it.

Three independent control systems — rotary encoder, colour sensor, and force-sensitive resistor — each handle one aspect of arm behaviour.
01

Arm Structure

Built the physical arm structure in LEGO Technic, designing the joint geometry for smooth rotation and a functional gripper mechanism.

02

Wheel-Motor Joint Control

Implemented wheel-motor interface for the main joint rotation — turning a wheel physically drives the arm joint, with encoder feedback for position control.

03

Colour Sensor Height Adjustment

Mounted a colour sensor to detect floor surface colours — different colours map to different target heights, enabling pre-programmed positioning.

04

Force-Sensitive Gripper

Integrated FSR sensors in the gripper fingers — the gripper closes until a target force is detected, then holds. Prevents crushing objects.

05

Arduino Integration

Wrote the unified control firmware that coordinates all three sensing modalities simultaneously without conflicts.
